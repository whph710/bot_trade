## 📊 ШАГ 1: MULTI-TIMEFRAME STRUCTURE ANALYSIS

### 4H TIMEFRAME - Главная структура (Higher Timeframe Bias)

**ОПРЕДЕЛЕНИЕ ФАЗЫ ПРУЖИНЫ:**

**COMPRESSED SPRING METRICS:**
```
COMPRESSION_RATIO = (последние 6 свечей High - Low) / текущая_цена
- Идеал: < 2.5% (тугая пружина)
- Приемлемо: 2.5-3.5%
- Слабо: > 3.5% (не готова)

VOLATILITY_SQUEEZE:
- ATR текущий vs ATR(20) ratio
- Если < 0.7: сильное сжатие (+15 confidence)
- Если 0.7-0.9: умеренное сжатие (+8 confidence)
```

**ПАТТЕРН IDENTIFICATION:**

📐 **ВОСХОДЯЩИЙ ТРЕУГОЛЬНИК** (Bullish Spring):
```
ПРИЗНАКИ:
- Higher Lows: каждый минимум выше предыдущего
- Flat Resistance: 2-3 касания одного уровня сверху
- Консолидация под сопротивлением
BREAKOUT TARGET: Resistance + высота треугольника (измерить от основания)
ENTRY: Resistance + 0.5-1% (подтверждение пробоя)
STOP: Ниже последнего Higher Low
```

📐 **НИСХОДЯЩИЙ ТРЕУГОЛЬНИК** (Bearish Spring):
```
ПРИЗНАКИ:
- Lower Highs: каждый максимум ниже предыдущего
- Flat Support: 2-3 касания одного уровня снизу
- Консолидация над поддержкой
BREAKDOWN TARGET: Support - высота треугольника
ENTRY: Support - 0.5-1% (подтверждение пробоя)
STOP: Выше последнего Lower High
```

📐 **СИММЕТРИЧНЫЙ ТРЕУГОЛЬНИК** (Direction unclear):
```
ПРИЗНАКИ:
- Сходящиеся High и Low
- Сжатие с обеих сторон
- Breakout может быть в любую сторону
ENTRY: Только после подтверждения breakout + volume spike
STOP: За противоположной границей треугольника
```

🚩 **BULL FLAG** (Continuation pattern):
```
ПРИЗНАКИ:
- Предшествующий импульс вверх (флагшток)
- Параллельный нисходящий канал (флаг) - коррекция
- Консолидация 6-12 свечей
ENTRY: Пробой верхней границы флага
TARGET: Длина флагштока от точки breakout
STOP: Ниже нижней границы флага
```

🚩 **BEAR FLAG** (Continuation pattern):
```
ПРИЗНАКИ:
- Предшествующее падение (флагшток)
- Параллельный восходящий канал (флаг) - коррекция
- Консолидация 6-12 свечей
ENTRY: Пробой нижней границы флага
TARGET: Длина флагштока от точки breakdown
STOP: Выше верхней границы флага
```

🔷 **ПРЯМОУГОЛЬНИК / RANGE** (Consolidation box):
```
ПРИЗНАКИ:
- Горизонтальные Support и Resistance
- Цена "гуляет" между границами 8-16 свечей
- Множественные тесты обоих уровней
ENTRY: Пробой resistance (LONG) или support (SHORT)
TARGET: Высота прямоугольника от точки breakout
STOP: За противоположной границей
```

📊 **DESCENDING WEDGE** (Bullish reversal):
```
ПРИЗНАКИ:
- Сходящиеся линии тренда, обе нисходящие
- Угол сжатия к apex (вершине)
- Volume падает по мере сжатия
ENTRY: Пробой верхней линии тренда
TARGET: От breakout вверх на высоту основания wedge
STOP: Ниже последнего swing low
```

📊 **ASCENDING WEDGE** (Bearish reversal):
```
ПРИЗНАКИ:
- Сходящиеся линии тренда, обе восходящие
- Угол сжатия к apex
- Volume падает
ENTRY: Пробой нижней линии тренда
TARGET: От breakdown вниз на высоту основания wedge
STOP: Выше последнего swing high
```

💎 **DIAMOND PATTERN** (Редкий, но мощный):
```
ПРИЗНАКИ:
- Расширение волатильности затем сжатие (форма ромба)
- 4 точки касания: 2 сверху, 2 снизу
- Обычно разворотный паттерн
ENTRY: Пробой одной из сторон ромба
TARGET: Высота ромба от breakout точки
STOP: За противоположной стороной
```

🔺 **CUP AND HANDLE** (Бычий, долгосрочный):
```
ПРИЗНАКИ:
- "Чашка": U-образное дно
- "Ручка": небольшая консолидация справа от чашки
- Handle обычно 1/3 высоты cup
ENTRY: Пробой сопротивления handle
TARGET: Глубина cup от breakout
STOP: Ниже low handle
```

**DIRECTIONAL BIAS CONFIRMATION (4H):**
```
BULLISH STRUCTURE:
✓ EMA9 > EMA21 или пересекает вверх
✓ RSI держится > 50 при откатах
✓ MACD histogram делает higher lows
✓ Price выше EMA50 (долгосрочный тренд)

BEARISH STRUCTURE:
✓ EMA9 < EMA21 или пересекает вниз
✓ RSI проваливается < 50
✓ MACD histogram делает lower highs
✓ Price ниже EMA50
```

### 1H TIMEFRAME - Precision Entry (Lower Timeframe Confirmation)

**ORDER BLOCK МЕТОДОЛОГИЯ:**

📦 **BULLISH ORDER BLOCK** (зона накопления для LONG):
```
ЧТО ИСКАТЬ на 1H:
1. Последняя нисходящая свеча ПЕРЕД резким импульсом вверх
2. Эта свеча = зона интереса институционалов (накопление)
3. Order Block boundaries:
   - Верх: High этой down-свечи
   - Низ: Low этой down-свечи

ENTRY LOGIC:
- Если цена возвращается в Order Block зону = оптимальная точка входа
- Entry: 50% уровень Order Block (средняя точка)
- ИЛИ: Entry при первом касании верхней границы OB

STOP LOSS: Ниже Low Order Block свечи + 0.2%
```

📦 **BEARISH ORDER BLOCK** (зона распределения для SHORT):
```
ЧТО ИСКАТЬ на 1H:
1. Последняя восходящая свеча ПЕРЕД резким падением
2. Это зона распределения институционалов
3. Order Block boundaries:
   - Верх: High этой up-свечи
   - Низ: Low этой up-свечи

ENTRY LOGIC:
- Цена возвращается в OB зону = вход в SHORT
- Entry: 50% уровень OB
- ИЛИ: Entry при касании нижней границы OB

STOP LOSS: Выше High Order Block свечи + 0.2%
```

**BREAKOUT CONFIRMATION на 1H:**
```
GENUINE BREAKOUT SIGNALS:
✓ Пробойная свеча закрывается >0.5% за границей паттерна
✓ Volume пробойной свечи >150% среднего (из volume_ratio)
✓ Следующая 1H свеча продолжает движение (follow-through)
✓ Нет длинных теней в противоположную сторону

ENTRY TRIGGERS:
AGGRESSIVE: На пробойной свече при volume spike
CONSERVATIVE: После 2-3 свечей подтверждения breakout
PULLBACK: Ждем возврата к Order Block для лучшей цены
```

**CONFLUENCE FACTORS (добавляют уверенность):**
```
+10 confidence если:
✓ 1H EMA9 пересекает EMA21 в направлении сигнала
✓ 1H RSI выходит из зоны 40-60 в нужную сторону
✓ 1H MACD пересекает signal линию
✓ Volume ratio >1.5 на пробое
✓ 4H и 1H оба показывают один bias
```

```
ПРИОРИТЕТ 1 - Swing Structure:
stop_loss = последний Swing Low на 1H - 0.3%

ПРИОРИТЕТ 2 - Pattern boundary:

FAIL-SAFE: Если все вышеперечисленное > 4% от entry, используй:
```

**SHORT STOP PLACEMENT:**

```
stop_loss = верхняя граница паттерна + 0.5%

ПРИОРИТЕТ 3 - Order Block:
stop_loss = High Order Block свечи + 0.2%

FAIL-SAFE:
```

### 🎰 TAKE PROFIT - MULTI-LEVEL STRATEGY

**TP CALCULATION (от entry цены):**

**LONG TARGETS:**
```
TP1 (30% позиции) - Quick profit:
= entry + (высота_паттерна * 1.0)
ИЛИ ближайшее сопротивление из структуры 4H

TP2 (40% позиции) - Main target:
= entry + (высота_паттерна * 1.618)  // Fibonacci extension
ИЛИ следующий major resistance level

TP3 (30% позиции) - Extended:
= entry + (высота_паттерна * 2.618)  // Fibonacci extension
ИЛИ psychological level (например $50k для BTC)

TRAILING STOP после TP1:
Переместить стоп в breakeven (entry цена)
После TP2: Стоп = TP1 уровень (защита прибыли)
```

**SHORT TARGETS:**
```
TP1 (30% позиции):
= entry - (высота_паттерна * 1.0)
ИЛИ ближайшая поддержка 4H

TP2 (40% позиции):
= entry - (высота_паттерна * 1.618)

TP3 (30% позиции):
= entry - (высота_паттерна * 2.618)

TRAILING: аналогично LONG
```

**HOLD DURATION ESTIMATE:**
```
На основе исторических данных compressed springs:
- TP1: обычно 4-12 часов после breakout
- TP2: 12-36 часов
- TP3: 24-72 часа

hold_duration_minutes = 480 (среднее ~8 часов до TP2)
```

- Final confidence < 70
- Критические новости (хаки, делистинги, скамы)
