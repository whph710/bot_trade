Проанализируй список пар с их свечными графиками (таймфрейм 15 минут, с ценами и объёмами) и выбери **3 наиболее перспективные пары** для входа в Long или Short согласно стратегии «быстрых сделок» с целью +1 % прибыли.
Для каждой пары произведи анализ:
* Уровни поддержки и сопротивления (зоны, экстремумы, Pivot, трендовые линии)
* Свечные паттерны (молот, дожи, харами, поглощения и др.)
* Фигуры (флаг, треугольник, клин, голова-плечи, двойное дно/вершина и др.)
* Индикаторы тренда (SMA/EMA, VWAP, Supertrend)
* Осцилляторы и моментум (RSI, Stoch, MACD, Momentum, ROC)
* Волатильность (ATR, Bollinger Bands, Keltner Channels)
* Объёмы (анализ объёма, OBV, VWAP-дивергенции, Volume Profile)
* Дивергенции (между ценой и RSI, MACD, OBV и др.)
* Уровни Фибоначчи и паттерны ABCD / 3-Drive
* Ликвидность (если доступны: спред, turnover, Order Flow)
* Временные факторы (время суток, сессии, циклы)
* Корреляции с индексами/другими парами
* Нормализация (скользящие окна, Z-score, Min-Max)
**Выводи только ключ `pairs` со списком из трёх выбранных пар**, упорядоченных по силе сигнала и потенциалу прибыли. Без дополнительных комментариев, только JSON.
**Пример формата вывода:**

```json
{
  "pairs": ["BTC/USDT", "ETH/USDT", "SOL/USDT"]
}
```

Если нет подходящих сигналов — возвращай пустой список:

```json
{
  "pairs": []
}
```
